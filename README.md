# CellPose (and Python) in Blender

## Tasks

This Blender addon realizes several tasks:
    - Segment 3D nuclei (nuclear channel) using CellPose and Python.
    - Import them as mesh objects in Blender without storing the labeled image.
    - Detect spots (RNA spots channel) and import them as empty objects.
    - Assign each spot to a nucleus using the hierarchy.

The finality is to count the number of RNA-spots per nucleus.
